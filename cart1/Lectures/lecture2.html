Categorical


Comparison of categories or distributions of quantitative values. These are often characterized by size.

Bar Charts- displays quantitative vaules of different categories as bar height (size)

Dot Plot- displays quantitative vaules of different categories as dots (size)

Pictograms- displays quantitative values of different categories as size and type of glyph

Proportional Symbol Chart- displays quantitative vaules of different categories as circles (size) and also uses color to differentiate additional categorical information.

Radial Graph- displays quantitative values as radius length.

Range Chart- displays the distribution of data values for categories (box and whisker-plots also quantiles)

Histogram- displays freqency and distribution of quantitative groups

Word Clouds- frequency of individual word frequencies by size

Hierarchical


Part-to-whole relationships

Waffle Chart- quantites of categories that are part of a whole. Number of associated sqaures show proportions

Stacked Bar Chart- displays part-to-whole using proportion of the whole bar.

Back to Back Bar Chart- The length of the bar is used to show proprtions for for two diverging categories (positive and negative usually)

Venn Diagram- shows relationships between collections and multiple sets

Node-Link Diagram- shows hierarchical relationships between multiple tiers

Relational


Correlations and Connections

Scatterplot- shows the relationship between two quantitative values

Parrallell Coordinate Plot- display multiple relationships between quantitative attributes

Heatmap- shows the quantitative value shared between two variables (frequencies)

Temporal

Time Trends

Line Chart- shows how quantitative values change over time. The use of a line denotes relationship between the indiviual values

Slope Graph Chart- shows the values and the slope between them for two points in time.

Gantt Chart- displays the steps of a project with each step/activity represented as section of bar

Spatial


mapping spatial patterns

Benefits of maps

  • They help us visualize large amounts of data more easily than numeric representation.
  • Visualizing allows the perception of patterns not readily visible at the indvidual level.
  • Maps can allow for the visualize small-scale and large-scale properties simultaneously.
  • Visualization allows for data exploration and hypothesis formation.
  • They help us tell stories.

  • Reference Maps
  • Thematic Maps

The limitations of maps (Tyner 2010)

  • Maps are selective and symbolic.
  • Maps are not without bias.
  • Maps require scaling.
  • Maps are distorted through projection.
  • Maps are only useful for spatial information.

Cartograms

Use of area and/or shape to differentiate between quantitative values

Things to Consider

  • Normalize your variable.
  • They distort real boundaries making it harder to read.
  • There are multiple types based on modification of shape or area, and the continuity of polygons.

Choropleth Map

Color value is used to denote differences in quantitative variable

Things to Consider

  • Normalize your variable.
  • Classification is your friend.
  • Color or pattern is used to differntiate between levels

Continous Surface Map

Continous representation of variable based on samples of the variable across space

Things to consider

  • Map changes of a continuous variable that may be sampled
  • Surface maps typically are the result of point pattern analyses
  • Hot-Spot Color Combinations (red-blue) should only be used for Hot-Spot Maps

Daysymmetric Map

Map based on statistcally-defined divisions instead of purely political boundaries.

Things to Consider

  • Secondary geographic patterns are used reflect more accurate distributions
  • It can be used with both choropleth or dot density maps
  • The map does end up emphasizing this secondary distribution

Dot Density Maps

Map using points to show the density of a phenomenon

Things to Consider

  • Distribution is not based on absolute location
  • Dots can represent 1 or more phenomenon
  • Hue can be used to show categorical differences

Flow Map

Map representing the flow of phenomenon through space

Things to Consider

  • Can illustrate categorical or magnitude attributes of the flow
  • Types include: network, radial, distributive and continuous
  • Bivariate flow maps use hue and line width to represent multiple attributes

Graduated Symbol Maps

Symbol size is used to represent quantitative values

Things to Consider

  • Map symbol size represents a range of values
  • Used for data in ratio or interval scale of measurement
  • Occlusion due to overlapping symbosl can be a problem

Isarithmic Map

2-D representation of 2-D phenomenon

Things to Consider

  • Isometric- constructed from sampled points
  • Isopleth- show how features differ in quantity as a surface.
  • Contours are labeled and fill can be used between contours

Isochrome Map

Representation of time/distance away from a feature using lines of equal value

Things to Consider

  • Circles can denote time or distance
  • Time and distance can be displayed together by adding color
  • Alternative to radial overlay, you can use color coded-routes

Proportional Symbol Map

Symbol size represents quantitative differences

Things to Consider

  • Symbol size differences should be easily perceptible
  • Good for interval or ratio level of measurement
  • Symbol size is typically established using the aqure root of the data value

Voroni Maps

Partition of a plane into regions based on distance to points in a subset of the main data

Things to Consider

  • Areal representation of the closest by proximity of points are located
  • May not be conceptually sound based on the feature you are trying to map
  • Larger shapes represent groups making up a larger proportion of the population
