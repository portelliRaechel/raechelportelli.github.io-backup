Symbol Maps

It's all about size

The map symbols size varies based on the data

  • Proportional - map symbol size is proportional to the data. Continous Change!
  • Graduated - map symbol size is classed. Not continous change in symbol size!

Proportional Symbol Map

Proportional Symbol Map

When to use these maps?

  • When a dot density map of the data would be too occluded
  • Symbolization of totals over an area
  • Symbolization of totals for a specific places
  • When comparing subdivisions

When not to use these maps?

  • Interval level data (this is arguable)
  • Symbolization of density
  • Data with small data ranges

Considerations

  • Ratio and interval data can be used in these maps
  • Can be used to to locate specific place values better than dot density maps
  • Enumeration unit size doesn't matter so you do not need to normalize your data

More Considerations

  • Make sure differences are easily distinguishable.
  • Overlap is inevitable, but transparency can help
  • For GS cases, make sure all classes are present in the legend
  • Simple shapes are best so that comparison is easier

Symbol Size Scaling

Perception of symbol area is difficult for humans.

Symbol Size Scaling: Proportional Symbol Maps

Mathematical: symbol size = (maximum symbol size) * (square root / max square root)

Perceptual: uses a power function

Symbol Size Scaling: Perceptual

How do I choose?

  • Very large numbers, outliers, and large data ranges can make the symbology cumbersome and overwhelm smaller values.
  • In those cases, using graduated symbols instead of proportional ones can alieviate some issues
  • For outlier cases, a single symbol can be used to represent values outside of a prefered threshold

Cartograms

size/area of enumeration unit is changed to reflect data source.

Use: reduces bias created by aggregating data by regions in choropleth map as areas with more data are shown larger than areas with less/smaller data

Note: sometimes it can be so deformed it is difficult to undertand the underlying geography
